# rzkfold

**An effortless way to scaffold python, rust, and web projects.**

---

## Introduction

**rzkfold** *(pronounced "razk-fold")* is a CLI app made using [vecli](https://lib.rs/crates/vecli)
and aims to be an easy way to scaffold multiple projects.

Being made in Rust, rzkfold gains all the performance boosts without costing any UX.
