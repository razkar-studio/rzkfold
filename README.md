# rzkfold

**An effortless way to scaffold python, rust, and web projects.**

---

## Introduction

**rzkfold** *(pronounced "razk-fold")* is a CLI app made using [vecli](https://lib.rs/crates/vecli)
and aims to be an easy way to scaffold multiple projects.

Being made in Rust, rzkfold gains all the performance boosts without costing any UX.

> [!WARNING]
> rzkfold is under construction and has only the bare minimum. This was made on Android in my phone as a project written using the Vecli rust crate.
> This README is also under construction.

## Usage

Install rzkfold:

```sh
cargo install rzkfold
```

Make sure you installed it correctly:

```sh
rzkfold --version
```

Scaffold a new project with the name "razkar":

```sh
rzkfold new razkar
```

Answer the interactive prompts. A new directory should be made.

Congrats! You made a project using rzkfold.
